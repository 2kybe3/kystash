# kystash

A simple image/file sharing server and client.

> **Disclaimer:** This project is in its early stages. Don't expect anything to work yet.

## Installation

### Nix

#### Profile
```bash
nix profile install git+https://git.kybe.xyz/2kybe3/kystash
```

#### One time run
```bash
nix run git+https://git.kybe.xyz/2kybe3/kystash
```

### Cargo

#### Unstable
```bash
cargo install --git https://git.kybe.xyz/2kybe3/kystash
```
