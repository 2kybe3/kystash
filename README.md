# kystash

**Kystash** is a file sharing server and client that aim's to be simple and efficient.

 - Chunked uploading that can be resumed at any time
 - A simple storage format that is easy to maintain


> **Disclaimer:** This project is in its early stages. Do not expect full functionality yet.

## Installation

### Nix

#### Shell

```bash
nix develop git+https://git.kybe.xyz/2kybe3/kystash#kystash
```

#### Profile
```bash
nix profile install git+https://git.kybe.xyz/2kybe3/kystash
```

#### One time run
```bash
nix run git+https://git.kybe.xyz/2kybe3/kystash
```

### Cargo

#### Unstable
```bash
cargo install --git https://git.kybe.xyz/2kybe3/kystash
```
